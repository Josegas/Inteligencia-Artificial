El presente trabajo tiene como objetivo documentar la construcción de las reglas y hechos que conforman la base de conocimiento de un sistema experto enfocado en la orientación vocacional. Este sistema tiene como finalidad identificar las áreas profesionales más adecuadas para un usuario, a partir de sus gustos, intereses, habilidades y preferencias personales.

La estructura del trabajo se basa en una clasificación organizada por áreas del conocimiento, como: Ciencias de la Salud, Ingeniería y Tecnología, Ciencias Económico-Administrativas, Ciencias Sociales y Humanidades, Artes y Diseño, Lenguas y Comunicación, Educación y Pedagogía, Ciencias Naturales y Ambientales, Turismo, Gastronomía y Servicios, y Ciencias del Deporte y Actividad Física.

Para cada área se definieron proposiciones simples que representan características o intereses particulares (como "te interesa el cuerpo humano", "te gusta trabajar con números", "te interesa la cocina", etc.). Estas proposiciones se codificaron mediante letras para facilitar la formulación de reglas lógicas.

Cada conjunto de proposiciones, cuando se cumple, determina la pertenencia del usuario a un área específica. Posteriormente, se añadieron reglas adicionales que, al combinar los intereses particulares dentro de un área, sugieren carreras universitarias concretas como Medicina, Ingeniería Civil, Psicología, Diseño Gráfico, Derecho, Gastronomía, Educación Especial, entre muchas otras.

Las reglas están escritas en formato de lógica proposicional, lo que permite su futura implementación en un lenguaje como Prolog. Asimismo, las letras asignadas se reinician en cada área para mantener claridad y orden lógico dentro de cada conjunto de reglas.

Este trabajo representa un componente fundamental dentro del diseño del sistema experto, ya que la base de conocimiento es el núcleo que guía el razonamiento del sistema para emitir recomendaciones vocacionales fundamentadas. La elaboración detallada y organizada de estas reglas y hechos permitirá, en etapas posteriores, su integración con un motor de inferencia y su representación gráfica mediante diagramas.
