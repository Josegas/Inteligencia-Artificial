Tarea: "Adquisición y Preprocesamiento de Imágenes para la Clasificación de Emociones"

Este proyecto tiene como finalidad preparar el dataset FER-2013 para ser utilizado en un sistema de reconocimiento de emociones. Este conjunto de datos contiene miles de imágenes de rostros etiquetados con una emoción específica como felicidad, tristeza, enojo, miedo, sorpresa, asco o neutral.

Para organizar este material, usamos un código en Python que extrae las imágenes del archivo .csv, las convierte desde cadenas de texto a imágenes reales en escala de grises, y las guarda en carpetas separadas según la emoción que representan. También utilizamos la herramienta FiftyOne para visualizar el conjunto ya procesado y verificar que las imágenes estén bien clasificadas.

Con este preprocesamiento, buscamos que los datos estén limpios, organizados y listos para entrenar un modelo de inteligencia artificial. El proyecto no incluye el entrenamiento del modelo, solo la parte de adquisición y preparación de los datos. Usamos herramientas gratuitas y accesibles como Python, OpenCV y FiftyOne, lo que lo hace un proyecto práctico, económico y útil para tareas relacionadas con el análisis de emociones mediante visión por computadora.
